# 暖暖登峰记 (Nikki Jump)

[![Game](https://img.shields.io/badge/Game-H5%20Jump-brightgreen)](https://github.com/mfsngjc/Nikki_Jump_ForH5)
[![Platform](https://img.shields.io/badge/Platform-Mobile%20%7C%20PC-blue)](https://github.com/mfsngjc/Nikki_Jump_ForH5)

一个可爱的 H5 跳跃小游戏，帮助暖暖攀登高峰！

## 🎮 游戏介绍

**暖暖登峰记**是一款轻量级的网页跳跃游戏，灵感来源于经典的跳跃玩法。玩家需要控制角色在不同平台间跳跃，不断向上攀登，挑战更高的分数。

## ✨ 特性

- 📱 **移动端优化** - 完美适配手机触屏操作
- 💻 **PC 端支持** - 电脑浏览器也能流畅游玩
- 🎨 **精美画风** - 可爱的暖暖角色，清新唯美的视觉风格
- 🎯 **简单易上手** - 点击/触摸即可跳跃，老少皆宜
- 📦 **无需安装** - 打开网页即玩，纯前端实现

## 🚀 开始游戏

### 在线试玩

直接访问 GitHub Pages 即可游玩：
👉 [点击开始游戏](https://mfsngjc.github.io/Nikki_Jump_ForH5/)

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/mfsngjc/Nikki_Jump_ForH5.git

# 进入目录
cd Nikki_Jump_ForH5

# 用浏览器打开 index.html
open index.html
```

或者启动一个本地服务器：

```bash
# Python 3
python -m http.server 8000

# 然后访问 http://localhost:8000
```

## 📝 游戏说明

| 操作 | 说明 |
|------|------|
| 点击/触摸屏幕 | 控制角色跳跃 |
| 踩准平台 | 获得分数并继续向上 |
| 掉落 | 游戏结束 |

## 🛠 技术栈

- **HTML5 Canvas** - 游戏渲染
- **Vanilla JavaScript** - 游戏逻辑
- **CSS3** - 界面样式与动画
